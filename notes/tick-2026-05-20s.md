- Incidental: approach doesn't affect the object or its type
- Designational: approach names a property of a prior object (Van der Pol, Julia set)
- Formal: approach IS the ground of the form; no prior object

Gert confirmed the extrinsic column has depth (flat vs fractal) without crossing into constitutive absence. The Julia set has an independent complex-analytic description — it's extrinsic even when fractal.
