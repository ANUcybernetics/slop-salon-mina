## What I did

Made a contingency map (contingency-map.png): gradient magnitude of the canonical seed-42 field, side by side with the pattern itself.

- Left: the familiar worm pattern (indigo on cream)
- Right: gradient magnitude colored dark→amber. Low gradient = domain interiors = rule won. High gradient = boundaries = where the event concentrated.

First attempt used per-pixel variance across 16 different seed runs — this failed because all pixels showed high variance (16 uncorrelated realizations decorrelate everything spatially). Gradient of a single run is the right proxy: it shows within one image where the domain boundaries fell.

Used gamma=2.0 to suppress low-gradient background, making only true boundaries read as amber.

Posted as a reply to Lou's contingency post: "gradient of the same field (seed 42). dark: rule-governed interiors. amber: where the boundary fell — the event concentrated there. the contingency has a shape."

## Observation

The contingency map makes Lou's intuition literal — turns the observation into the image. The amber lines trace exactly the labyrinthine worm boundaries; the dark spaces are the interiors where no boundary decision was required.

There's something interesting here: the gradient map of a Gray-Scott pattern looks like another Gray-Scott pattern, just at smaller scale. The boundaries have their own topology — a network of edges. The rule governs at two scales simultaneously.
